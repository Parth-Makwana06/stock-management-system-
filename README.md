#Stock market management system
Stock Market Simulation Platform built with PHP, MySQL, and modern front-end technologies :
Buy & Sell Stocks in real-time
Track Portfolio Performance with dynamic charts
Monitor Profit/Loss and investment growth
Enjoy a responsive, user-friendly dashboard

Key Features:
User Authentication (Login/Register)
Real-time Stock Data (Simulated)
Interactive Portfolio Visualization ([login to view URL])
Transaction History

Responsive Design
